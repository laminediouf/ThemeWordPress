== Description ==
Oneway is a child theme of zerif lite. It contain sections with gradient overlay. Which enhance  beauty of zerif. This design is suitable for a website of your desired niche (Like Agency, Personal, Local business, Corporate etc). Additionally it contain all options available with zerif lite theme.
